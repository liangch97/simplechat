# 🎓 SYSU Chat 学习指南

欢迎来到 SYSU Chat 项目学习模块！本指南将帮助你从零开始理解这个聊天室项目。

## 📚 学习路线

建议按以下顺序学习：

| 序号 | 模块 | 文件夹 | 难度 | 状态 |
|------|------|--------|------|------|
| 1 | Java 网络编程基础 | `01-network-basics/` | ⭐ | ✅ 完成 |
| 2 | HTTP 服务器原理 | `02-http-server/` | ⭐⭐ | ✅ 完成 |
| 3 | SSE 实时通讯 | `03-sse-realtime/` | ⭐⭐⭐ | ✅ 完成 |
| 4 | 前端 JavaScript | `04-frontend-js/` | ⭐⭐ | ✅ 完成 |
| 5 | 完整项目实战 | `05-full-project/` | ⭐⭐⭐ | ✅ 完成 |

### 模块内容概览

- **模块1**: Socket/ServerSocket 基础、TCP 通信、多线程处理
- **模块2**: HTTP 协议、com.sun.net.httpserver、路由处理
- **模块3**: SSE 原理、EventSource、实时推送
- **模块4**: DOM 操作、Fetch API、EventSource API、事件处理
- **模块5**: 项目架构分析、代码追踪、扩展练习

## 🛠 学习环境准备

1. 安装 JDK 11 或更高版本
2. 安装 VS Code 或 IntelliJ IDEA
3. 安装浏览器开发者工具 (F12)

## 📝 如何使用练习代码

每个模块文件夹包含：
- `README.md` - 知识点讲解
- `examples/` - 示例代码
- `exercises/` - 练习题目
- `solutions/` - 参考答案

## 🚀 开始学习

从 [01-network-basics](./01-network-basics/README.md) 开始你的学习之旅！
